# OpenAI plugin

## 功能

- [x] 支持 chatgpt



## 快速开始

### 1. 添加 Maven 依赖

```xml
<dependency>
    <groupId>com.saucesubfresh</groupId>
    <artifactId>open-starter-openai</artifactId>
    <version>1.0.0</version>
</dependency>
```

### 2. 相关配置

```yaml
com:
  saucesubfresh:
    openai:
      
```
